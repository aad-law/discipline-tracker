# Discipline Tracker

A full-stack habit tracking app.

## Tech Stack
- Frontend: React
- Backend: Java Spring Boot
- Database: PostgreSQL

## Setup Instructions

### Backend
1. Go to `backend/` folder
2. Copy `src/main/resources/application.properties.example`
3. Rename it to `application.properties`
4. Fill in your PostgreSQL credentials
5. Run the app in Eclipse

### Frontend
1. Go to `frontend/` folder
2. Run `npm install`
3. Run `npm run dev`
